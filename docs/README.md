## 문제 흐름에 따른 이해

1. 로또 금액입력
  - 숫자 확인
  - 1000원 단위로 금액 확인

2. 로또번호 구매 수량만큼 번호 출력
  - 1~45까지의 숫자 확인
  - 중복된 숫자 확인
  - 오름차순으로 정렬

3. 당첨번호 입력
  - 6개의 숫자입력
  - 숫자 확인
  - 1~45까지의 숫자 확인
  - 중복된 숫자 확인

4. 보너스번호 입력
  - 숫자인지 확인
  - 1~45까지의 숫자확인
  - 중복 확인

5. 통계 구하기
   1) 3개 일치 (5,000원)
   2) 4개 일치 (50,000원)
   3) 5개 일치 (1,500,000원)
   4) 5개 일치, 보너스 볼 일치 (30,000,000원)
   5) 6개 일치 (2,000,000,000원)

6. 수익률 구하기
  - 소수점 둘째 자리에서 반올림
